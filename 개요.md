# LAN Audio Relay

Windows PC에서 재생 중인 소리를 같은 Wi-Fi/LAN에 있는 다른 Windows PC에서 실시간으로 들을 수 있게 해주는 간단한 오디오 공유 프로그램입니다.

## 주요 기능

- PC 전체 재생 소리 전송
- 같은 LAN 안에서 자동 검색 또는 IP 직접 연결
- 6자리 페어링 코드로 연결 확인
- 낮은 지연을 위한 Opus 오디오 전송
- 별도 설치 없이 실행 가능한 Windows용 EXE 제공

## 사용 방법

1. 소리를 들을 PC에서 `Receiver`를 시작합니다.
2. 표시된 페어링 코드를 확인합니다.
3. 소리를 보내는 PC에서 Receiver를 선택하거나 IP를 입력합니다.
4. 페어링 코드를 입력하고 `Start Streaming`을 누릅니다.

## 참고

같은 네트워크에 있어야 하며, Windows 방화벽에서 앱 통신 허용이 필요할 수 있습니다.
